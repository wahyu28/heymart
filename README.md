# Web Based Point Of Sale Application

Aplikasi Point of Sales yang digunakan untuk memanage kebutuhan market

## Teknologi
- Laravel
- Jquery Ajax
- Bootstrap 4

## Fitur Modul
- Kategori Produk
- Produk
- Member
- Supplier
- Pengeluaran
- Penjualan
- Pembelian
- Laporan
- Setting
- User / Pengguna (Admin & Kasir)
- Realtime Validation
